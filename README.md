# Requerimentos

* R >= 3.4.4
* SQLite3 >= 3.22.0

# Configuração

Primeiramente, deve criar o banco de dados SQL com o SQLite, rodando o script `src/dbcreate.sql`.

```sh
sqlite3 data/covid.db < src/dbcreate.sql
```
